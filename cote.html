Comment on calcule.

La Cote CARNET n'est pas un guide imprimé. C'est un agrégat vivant, recalculé chaque nuit sur la base réelle du marché européen. Chaque source a un poids transparent, chaque ajustement est documenté, et la formule complète est publique sur le GitHub de l'asso.

Réponses

Questions fréquentes

Ma voiture n'apparaît pas dans la liste

Si le modèle est trop singulier ou si l'échantillon est inférieur à 5 voitures actives, CARNET ne peut pas calculer de cote fiable. Tu verras alors la mention "Singulière" avec une suggestion vers un segment voisin à titre indicatif.

La cote Argus diffère beaucoup de CARNET

L'Argus se base sur un guide imprimé trimestriel. CARNET se base sur le marché vivant, recalculé chaque nuit. Pour les voitures récentes, ça converge. Pour les classiques, anciennes, ou très haut de gamme, ça peut diverger fortement — notre cote suit le marché réel, pas une formule théorique.

Je peux contester la cote ?

Oui. Si tu as des comparables précis (URL de vente BaT, AutoScout, enchères récentes), tu peux proposer une correction via le bouton "Contester" sur la page de résultat. Ta contribution sera examinée par la communauté experte du segment. Si adoptée, tu seras crédité publiquement.

C'est gratuit pour toujours ?

Oui. La Cote CARNET est un bien commun, jamais monétisée, jamais derrière paywall API. Aucune commission sur l'achat ou la vente. C'est l'asso qui la publie, dans l'intérêt général de la communauté automobile européenne.

La formule est-elle publique ?

Oui. Toute la méthodologie est documentée dans un repository GitHub public de l'asso, avec versioning commit-by-commit. Chaque modification fait l'objet d'une Pull Request publique commentée. Aucune boîte noire.

Mon assureur acceptera cette cote ?

Cinq assureurs partenaires (Carene, MAIF, Hagerty, Chubb, AXA Collection) acceptent la cote CARNET comme valeur de référence pour leurs polices. Pour les autres, tu peux l'utiliser comme point de négociation — c'est une référence factuelle, sourcée, vérifiable.
